true description: Migrate Data from Excel to Oracle 10g Express Edition meta: _wpas_done_twitter: '1' --- in my new chores, i had the need to export excel data to oracle database and i found one easy way to do this, well first we need to export our excel data to text file (.txt) delimited by pipes (|), you can check how to do in these link http://www.howtogeek.com/howto/21456/export-or-save-excel-files-with-pipe-or-other-delimiters-instead-of-commas/ so when you have your .txt file delimited by pipes you will get something like this {% highlight text %} luis|vasquez|levb20@gmail.com|123456|guatemala city luis|vasquez|levb20@gmail.com|7891011|guatemala city luis|vasquez|levb20@gmail.com|11121314|guatemala city luis|vasquez|levb20@gmail.com|14151617|guatemala city luis|vasquez|levb20@gmail.com|17181920|guatemala city luis|vasquez|levb20@gmail.com|2122232425|guatemala city {% endhighlight %} So the next step is add at the begin on the file this code {% highlight text %} LOAD DATA INFILE * INTO TABLE table_name APPEND FIELDS TERMINATED BY '|' OPTIONALLY ENCLOSED '"' TRAILING NULLCOLS (first_name, last_name, email, phone, address) BEGINDATA {% endhighlight %} so you get this :) {% highlight text %} LOAD DATA INFILE * INTO TABLE table_name APPEND FIELDS TERMINATED BY '|' OPTIONALLY ENCLOSED '"' TRAILING NULLCOLS (first_name, last_name, email, phone, address) BEGINDATA luis|vasquez|levb20@gmail.com|123456|guatemala city luis|vasquez|levb20@gmail.com|7891011|guatemala city luis|vasquez|levb20@gmail.com|11121314|guatemala city luis|vasquez|levb20@gmail.com|14151617|guatemala city luis|vasquez|levb20@gmail.com|17181920|guatemala city luis|vasquez|levb20@gmail.com|2122232425|guatemala city {% endhighlight %} and then you need to save this document with .ctl extention, finally start your CMD and go to the BIN folder in Oracle_Home installation, in my case this is my direction or path C:\oraclexe\app\oracle\product\10.2.0\server\BIN and run this {% highlight text %} sqlldr myuser/mypassword@mydb myfile.ctl rows=10 errors=30000 {% endhighlight %}
myuser: our instance user mypassword: Password db mydb: database oracle that we use myfile.ctl: my .ctl file rows=10 : COMMIT each 10 records errors=30000 : number of errors that can support
and that's it
